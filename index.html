CIDR & Subnet Calculator

What is CIDR?
CIDR (Classless Inter-Domain Routing) defines how many bits are used for network portion. Example: /24 means 24 bits network, 8 bits host.

Standard Values:
/24 = 256 IPs
/25 = 128 IPs
/26 = 64 IPs
IPv4 = 32 bits total

Binary / Decimal / Hex / Octal Converter

What is number system?
Networking uses binary (base-2). Humans use decimal (base-10). Hex is used in MAC addresses and memory addressing.

Standards:
Binary = base 2
Decimal = base 10
Hex = base 16
Octal = base 8

IP Range Calculator

What is IP Range?
It defines usable IP addresses inside a subnet. First IP = Network Address, Last IP = Broadcast Address.

dBm Converter (Power Level)

What is dBm?
dBm is a unit of power in telecom. It shows signal strength in fiber and wireless systems.

Standard: 0 dBm = 1 mW

Bandwidth Converter

What is Bandwidth?
Bandwidth is data transfer rate of a network. Higher Mbps = faster download speed.

Standard: 1 Mbps = 0.125 MB/s
