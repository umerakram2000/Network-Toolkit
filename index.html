Subnet Calculator

Definition: Subnetting divides a network into smaller logical networks using CIDR.

Standard Values:
IPv4 = 32 bits
/24 = 256 IPs
/25 = 128 IPs

Binary Converter

Definition: Binary is base-2 number system used in networking.

Standard: 1 byte = 8 bits

IP Range Calculator

Definition: Determines usable IP range inside a subnet.

Standard: First IP = Network, Last IP = Broadcast

dBm Converter

Definition: dBm measures optical/electrical power in telecom.

Standard: 0 dBm = 1 mW

Bandwidth Converter

Definition: Converts network speed to real download speed.

Standard: 1 Mbps = 0.125 MB/s
